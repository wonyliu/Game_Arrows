# 主界面设计知识库 V1 - 屏幕蓝图

## 1. 基准与约束
- 设计基准：`430 x 932`（竖屏）。
- 安全区：上 `24`、下 `24`、左右 `16`。
- 目标：在 `320x568 / 390x844 / 430x932 / 1024x768 / 1366x768` 下保持等比可点击。

## 2. HOME 蓝图
### 分区占比
- Top 品牌资源区：`y=24..202`（约 19.1%）
- Middle 主操作区：`y=214..560`（约 37.1%）
- Bottom 功能卡区：`y=576..892`（约 33.9%）

### 组件优先级
1. `home_primary_cta`（开始游戏）
2. `home_feature_*`（设置/排行榜/皮肤/签到/退出）
3. `home_level_tag`（当前关卡标签）
4. `home_resource_*`（金币/体力）

### 交互路径
- 点击 `home_primary_cta`：进入“已解锁最大关卡”。
- 点击 `home_feature_*`：打开对应面板。
- 点击 `home_level_tag`：进入关卡选择。

## 3. SETTINGS 蓝图
### 分区占比
- 面板头：约 14%
- 设置内容区：约 76%

### 核心模块
- 语言切换（`zh-CN / en-US`）
- 音频占位说明
- 画质占位说明

### 交互路径
- 返回按钮：回到 HOME。
- 语言切换：即时刷新标题与按钮文案。

## 4. LEADERBOARD 蓝图
### 核心模块
- 面板头
- 排行空态列表（5 行占位）
- 空态说明文案

### 交互路径
- 返回按钮：回到 HOME。

## 5. SKINS 蓝图
### 核心模块
- 面板头
- `3 x 2` 皮肤卡网格占位
- 空态说明文案

### 交互路径
- 返回按钮：回到 HOME。

## 6. CHECKIN 蓝图
### 核心模块
- 面板头
- `7` 天签到格占位
- 奖励说明占位

### 交互路径
- 返回按钮：回到 HOME。

## 7. EXIT_CONFIRM 蓝图
### 核心模块
- 面板头
- 确认说明文本
- 主按钮：确认退出
- 次按钮：取消

### 交互路径
- 确认退出：尝试关闭窗口；失败则显示反馈提示。
- 取消或关闭：回到 HOME。
